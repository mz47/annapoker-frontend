# Annapoker Frontend

## About
Planning Poker Web App helping agile Teams to estimate and refine User Stories

## Required

- RabbitMQ as STOMP Broker
- Corresponding Backend (Golang)
- ReThinkDB as Document Store

## Run
```shell script
npm start
```

## Build
```shell script
npm build
```

## Docker
```shell script
docker pull mz47/annapoker-frontend:latest
```
